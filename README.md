# Titanic Survival Prediction: Exploratory Data Analysis and Machine Learning

## Project Overview

This project applies Python-based data science techniques to analyze the famous Titanic dataset. The objective is to demonstrate practical competence in data acquisition, data cleaning, exploratory data visualization, statistical analysis, and introductory machine learning to predict passenger survival.

## Repository Contents

This repository contains the required files for the softcopy submission:

- `titanic_analysis.ipynb`: The primary Jupyter Notebook containing all Python code, visualizations, and modeling.
- `titanic.csv`: The raw dataset used for analysis (sourced from Kaggle).
- `Final_Report.pdf`: The complete written report discussing methodology, statistical insights, and machine learning results.
- `README.md`: This project documentation file.

## Technologies Used

The project was developed in a Jupyter Notebook environment using Python 3. The following libraries are required:

- **Pandas:** Data manipulation and cleaning.
- **NumPy:** Numerical operations.
- **Matplotlib & Seaborn:** Data visualization.
- **Scikit-Learn:** Machine learning (Logistic Regression) and performance metrics.

## How to Run the Code

1. Clone this repository to your local machine.
2. Ensure you have Jupyter Notebook or JupyterLab installed.
3. Install the required dependencies using pip:
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn
   ```
